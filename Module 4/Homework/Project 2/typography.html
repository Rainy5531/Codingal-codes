CSS

CSS is used to present and style a document written in a markup language such as HTML.

Using CSS, we can change the background, font, and do much more to make the webpage more attractive.

Example of CSS

Here are some examples of how CSS works:

                    class="text-success" style="font-size: 25px;"
                    class="text-danger" style="font-size: 30px;
